# Npz System Monitor
Npz System Monitor は、CPU, RAMの使用量/使用率、ディスクの読み書き速度、ネットワークの送受信量を表示するモニターアプリケーションです。

アプリケーションの動作には [**.NET Framework 4.6**](https://www.microsoft.com/ja-jp/download/details.aspx?id=17851) が必要です。
